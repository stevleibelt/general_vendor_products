# PC Engines APU2c4

## Serial console

* Install picocom
* `LANG=C picocom --baud 115200 /dev/ttyUSB`

## USB RS232

* Wire it
* Put usb into your usb
* screen on /dev/ttyUSB0

| Wire color | Function |
| --- | --- |
| red | 5v |
| black | gnd |
| white | RXD |
| green | TXD |
